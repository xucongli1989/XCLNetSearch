# 简介
asp.net 列表通用查询控件。本查询控件基于JS，提交Form，最终生成SQL中的条件语句，如"a=b or a=c and m like '%n%'"等
目前只支持SQL SERVER的语法规则。最近的更新中由原asp.net webform 服务器控件改为**直接输出string**，而不依赖于服务器控件载体。因此，本插件均支持webform和asp.net mvc。
